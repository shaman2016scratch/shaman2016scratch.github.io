Что такое SOVMS?

SOVMS это sosirsOS Version Manage System - система управления версиями sosirsOS. Это аналог GIT созданный для sosirsOS. Хотя название говорит "управлениями версий sosirsOS", система не для управления версиями sosirsOS, она просто создана для sosirsOS, и по этому содержит её название. Она так-же помогает управлять проектами.
