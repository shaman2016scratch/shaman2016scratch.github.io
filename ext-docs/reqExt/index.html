Request Information Documentation

What is it?

Request Information is an extension for Dash that allows you to find out information through a request.

What Api is used here?

It is used here https://api-shaman2016.vercel.app/req (Or else https://api-shaman2016.vercel.app/getMe)

What does each of the blocks mean and how do they work?

get User Agent

How it is used: Getting the browser's User Agent

How it works: uses the user-agent parameter in the api response

get Internet Protocol

How it is used: Getting the router's IP address (since IP is short for Internet Protocol)

How it works: Takes the "ip" parameter from the API response

get Request

How it is used: Get an API response

How it works: Gives the server response (received after let a = await fetch("https://api-shaman2016 .vercel.app/req") and await a.json())

get City

How it is used: Get the user's city (determined by Vercel by IP address)

How it works: Returns the header "x-vercel-ip-city"
